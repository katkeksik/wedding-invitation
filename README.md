# WEDDING — Andrey & Katya

Статичный сайт-приглашение. Открывается напрямую в браузере, работает офлайн, без сборки.

## Структура
```
index.html   — разметка страницы
style.css    — сброс, анимации, адаптив
script.js    — логика музыкального плеера (play/pause, seek, mute)
images/      — фото и текстуры
audio/       — фоновая музыка
```

## Публикация на GitHub Pages
1. Создайте новый репозиторий на GitHub и загрузите содержимое этой папки в корень репозитория.
2. В репозитории откройте **Settings → Pages**.
3. В разделе **Build and deployment** выберите **Deploy from a branch**, ветку `main` и папку `/root`.
4. Сохраните — через 1–2 минуты сайт будет доступен по адресу `https://<username>.github.io/<repo-name>/`.

## Музыка
Плеер внизу справа. Музыка начинает играть только после нажатия ▶ (без автовоспроизведения — так требуют браузеры), продолжает играть при прокрутке страницы, позиция воспроизведения сохраняется в localStorage.
